# Cheese Trials
This game was made for Campfire Cardiff 2026 using Godot, you play as a rat trying to find cheese while escaping the sewers! To get to the cheese, players must rotate pipes,and don't fall into the green acid below their feet!!

Surprisingly won first prize at Campfire Cardiff despite the amount of bugs the game has..!

# How to play?
You can control the rat by using WASD or arrow keys, pressing R will rotate the pipes.

# Difficulties?
This was made in a day by people who have barely any Godot experience, we had to learn a lot of things like player movement, scenes, ui etc in 12 hours (absolute madness)
